Crear una libreria que contenga un Code Template que sea capaz de convertir el sexo de un ADT_A01 en un tipo de dato FHIR

```
function convertGender(value) {
  // Codigo
}
```

Los datos hl7 v2 son:

* F : Female
* M: Male
* U: Unknow
* O: Other


Los datos FHIR para gender son (code/display)

* male: 	Male
* female:	Female
* other: 	Other
* unknown:	Unknown


Ref: https://hl7.org/fhir/R4/valueset-administrative-gender.html[https://hl7.org/fhir/R4/valueset-administrative-gender.html]
